To set this example up:

  1. Create an @anywhere applicaiton on dev.twitter.com (click here to get started). Double check the URL and be sure to upload an image.
  2. Replace the YOUR_API_KEY value at the top of this file
  3. Add the following HTML to your site: 
      <span id="follow-twittermedia"></span>
  <script type="text/javascript">
    twttr.anywhere(function (T) {
      T('#follow-twittermedia').followButton("twittermedia");
    });
  </script>
